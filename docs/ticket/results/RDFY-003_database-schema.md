## Notes
- Migration is `0000_initial.sql` (Drizzle Kit numbering convention) rather than `0001`. Spirit of the ticket (single initial migration from Drizzle Kit) is met.
- `PRAGMA journal_mode = WAL` is applied to the file-backed DB only — WAL is meaningless for `:memory:`.
- `PRAGMA foreign_keys = ON` is applied to both, so FK violations surface in tests too.
